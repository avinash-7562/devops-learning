# Git Fundamentals

This section contains my notes and hands-on practice while learning Git.

Git is a distributed version control system used to track changes in code and collaborate with others efficiently.

## Topics Covered

- What is Version Control
- What is Git
- Git Installation
- Git Configuration
- Basic Git Commands
- Git Workflow
- Branching and Merging
- Working with Remote Repositories

## Basic Git Workflow

1. Initialize or clone a repository
2. Make changes to files
3. Stage the changes
4. Commit the changes
5. Push to remote repository
